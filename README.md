# 🛒 Global Superstore Analysis | End-to-End Data Analytics

## 📌 Professional Summary

Developed an end-to-end data analytics solution to analyze global retail performance using SQL, Python, and Power BI. The project focuses on transforming raw transactional data into actionable insights to improve business decision-making, profitability, and customer retention.

---

## 🛠 Tech Stack & Competencies

* **Python:** Pandas, NumPy, Matplotlib, Seaborn
* **SQL:** PostgreSQL / MySQL (Joins, Subqueries, Aggregations)
* **Power BI:** DAX, Data Modeling, Interactive Dashboards
* **Excel:** Data Cleaning & Preprocessing

---

## 🚀 Key Project Phases

### 🔹 Data Extraction & SQL Analysis

* Performed complex SQL queries using joins and aggregations
* Calculated KPIs such as Total Sales, Profit, and Average Order Value

### 🔹 Exploratory Data Analysis (Python)

* Cleaned and processed dataset (50,000+ records)
* Identified trends, correlations, and outliers
* Visualized patterns using Matplotlib and Seaborn

### 🔹 Dashboard Development (Power BI)

* Built an interactive dashboard with KPI cards and filters
* Created DAX measures for performance tracking (YoY growth)
* Enabled dynamic filtering by Region, Segment, and Category

---

## 📈 Key Business Insights

* Identified top-performing regions contributing to maximum revenue
* Detected profit loss in specific product categories
* Segmented high-value customers for targeted strategies
